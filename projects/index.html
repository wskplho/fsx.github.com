Misaka

The Python binding for Sundown, a markdown parsing library. Misaka is made with Cython and C and provides the same/similar functionality as Redcarpet, the Ruby binding for Sundown. The parser and renderer are decoupled. This means you can make your own renderers.

  • Github
  • PyPi

Momoko

An asynchronous Psycopg2 wrapper for Tornado. Momoko makes use of the asynchronous support in Psycopg2, Tornado's IOLoop and gen to provide asynchronous queries in callback and blocking style.
